# Opertech - Gestione Cantieri

Applicazione web per gestione cantieri, risorse, rapportini e contabilità.

## 🚀 Deploy

Deploy automatico tramite Render usando Blueprint (`render.yaml`).

## 📦 Stack

- Next.js 14
- Prisma ORM
- PostgreSQL (Render)

## ⚙️ Comandi

```bash
npm install
npx prisma generate
npm run build
npm start
