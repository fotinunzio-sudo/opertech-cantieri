# Opertech - Gestione Cantieri MVP

Pacchetto completo Next.js + Prisma + PostgreSQL pronto per Render.

## Moduli inclusi
- homepage
- commesse
- risorse
- rapportini
- API REST base
- PostgreSQL con Prisma

## Variabile ambiente
Imposta su Render:
- DATABASE_URL=postgresql://...

## Build / Start Render
- Build Command: npm install && npm run build
- Start Command: npm run start
